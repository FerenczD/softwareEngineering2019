Project VI - About

The project is to design software to effectively control an elevator. The software will take advantage of CAN bus communication to deliver instructions from button presses to move the elevator to the specified floors. The elevator should indicate which floor its currently on by lighting up the corresponding LEDs. As an extra feature, users should be able to control and monitor the elevator remotely by using the elevator project website.
